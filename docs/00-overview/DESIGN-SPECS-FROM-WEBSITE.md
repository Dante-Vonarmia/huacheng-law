## 5. 设计元素

### 色彩方案（根据官网推测）

**主色系**：
- 深蓝/深灰色系（专业、稳重）
- 可能使用金色或黄色作为强调色（根据律所Logo）

**建议色彩规范**：
- **主色 (Primary)**：#2C3E50（深蓝灰）
- **辅色 (Secondary)**：#C9A961（金色）
- **强调色 (Accent)**：#3498DB（蓝色）
- **背景色**：#FFFFFF（白色）
- **次要背景**：#F5F7FA（浅灰）
- **文字主色**：#2C3E50（深灰蓝）
- **文字次色**：#7F8C8D（灰色）

### 字体（根据官网）

- **中文字体**：系统默认（建议使用思源黑体或苹方）
- **英文字体**：sans-serif（建议使用 Inter 或 Helvetica）

### 布局

- **容器宽度**：1170px（w1170-mg-auto 类名）
- **响应式**：支持移动端适配
